# Real-Time Conversational AI Agent

A real-time conversational AI assistant built using **LiveKit**, **Python**, and modern **LLM technologies**.
This project enables low-latency voice interaction, intelligent conversational responses, and modular agent-based automation suitable for real-time applications.

---

## Features

* Real-time voice interaction using LiveKit
* Speech-to-Text (STT) and Text-to-Speech (TTS) integration
* LLM-powered conversational intelligence
* Modular and scalable agent architecture
* Easy integration with external APIs and automation workflows

---

## Tech Stack

* Python
* LiveKit
* Large Language Models (LLMs)
* Speech-to-Text (Whisper / Deepgram / Google STT)
* Text-to-Speech (ElevenLabs / Azure TTS)

---

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/real-time-ai-agent.git
cd real-time-ai-agent
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Usage

Run the application:

```bash
python main.py
```

Make sure to configure your environment variables for API keys before running the project.
